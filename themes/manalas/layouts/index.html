The flexible, robust, and web oriented toolbox for Ansible.

Manalas is a manual to use Ansible roles created by Elao to handle our entire infrastructure, it provide configuration examples and best practices to fast build custom-designed infrastructures.

We choosed Ansible as IT automation tool regarding is simplicity and reliability. We are using it since a while now to build our clients infrastructures and we are convinced this is the way to go.
